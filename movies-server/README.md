# CRUD Movies Catalog Node

### Sescription

Simple RESTful CRUD APIs to do CRUD of a movie catalog

## Prereqisite

- Node v10+
- NPM v6+
- Mongodb Cloud Atlas cluster ( in this case we can use the hardcoded in .env demo file  )

## Getting Started

```
clone reposotory
npm install
cp env.demo .env.demo
npm start

```

## Usage
 
 npm start


## Dependencies Used
- express
- express-validator
- mongoose
